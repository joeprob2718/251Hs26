1. The Mass–Spring–Dashpot System

Consider a mass \(m\) attached to a spring and a dashpot (shock absorber). Let \(x(t)\) denote the displacement of the mass from its equilibrium position at time \(t\).

Forces acting on the mass:

The spring exerts a restoring force proportional to the displacement, directed back toward equilibrium:

\[ F_S = -kx, \qquad k > 0 \text{ (spring constant)}. \]

The dashpot exerts a resistive force proportional to the velocity and opposite to the direction of motion:

\[ F_R = -c\,\frac{dx}{dt}, \qquad c > 0 \text{ (damping constant)}. \]

Applying Newton's second law \(F = ma\):

\[ m\,\frac{d^2x}{dt^2} = F_R + F_S = -c\,\frac{dx}{dt} - kx. \]

Rearranging gives the homogeneous equation for free vibrations (also called a damped harmonic oscillator):

\[ \boxed{m\,\frac{d^2x}{dt^2} + c\,\frac{dx}{dt} + kx = 0.} \]

If an external force \(F(t)\) is also applied, we instead obtain the inhomogeneous equation for driven vibrations:

\[ m\,\frac{d^2x}{dt^2} + c\,\frac{dx}{dt} + kx = F(t). \]
Analogy: LRC Circuits

The same equation governs the current \(I(t)\) in an LRC-circuit: \[ L\,\frac{d^2I}{dt^2} + R\,\frac{dI}{dt} + \frac{1}{C}\,I = F(t), \] where \(F(t) = V'(t)\) is the derivative of the input voltage.

2. Characteristic Equation & Four Cases

The characteristic polynomial of \(m x'' + c x' + k x = 0\) is

\[ mr^2 + cr + k = 0, \]

with roots

\[ r_{1,2} = \frac{-c \pm \sqrt{c^2 - 4km}}{2m} = -\frac{c}{2m} \pm \frac{\sqrt{c^2 - 4km}}{2m}. \]

Since \(m, k > 0\) and \(c \ge 0\), the discriminant \(\Delta = c^2 - 4km\) determines four cases:

  • Undamped (\(c = 0\)): Roots \(r_{1,2} = \pm\sqrt{k/m}\,i\) are purely imaginary. The motion is an ideal oscillation with no friction.
  • Underdamped (\(0 < c < 2\sqrt{km}\)): Roots are complex with negative real part, since \(c^2 - 4km < 0\) and \(-c/(2m) < 0\). Damping is present but not enough to prevent oscillations.
  • Critically damped (\(c = 2\sqrt{km}\)): A repeated negative real root, since \(c^2 - 4km = 0\) and \(-c/(2m) < 0\). This is the boundary (bifurcation) between oscillatory and non-oscillatory behavior.
  • Overdamped (\(c > 2\sqrt{km}\)): Two distinct negative real roots, since \(c^2 - 4km > 0\) and \(\sqrt{c^2 - 4km} < c\). No oscillations occur.

3. Useful Trigonometric Identity

A linear combination of cosine and sine at the same angular frequency \(\omega\) can be written as a single shifted cosine:

\[ A\cos(\omega t) + B\sin(\omega t) = \sqrt{A^2 + B^2}\;\cos(\omega t - \alpha). \]

Here \(\tan\alpha = \dfrac{B}{A}\), with the phase angle chosen as:

\[ \alpha = \begin{cases} \arctan\!\left(\dfrac{B}{A}\right), & \text{if } (A,B) \text{ is in the 1st or 4th quadrant},\\[6pt] \arctan\!\left(\dfrac{B}{A}\right) + \pi, & \text{if } (A,B) \text{ is in the 2nd or 3rd quadrant}. \end{cases} \]

Key quantities:

  • Amplitude: \(\sqrt{A^2 + B^2}\)
  • Phase angle (delay angle): \(\alpha\)
  • Period: \(\dfrac{2\pi}{\omega}\)

This is precisely the polar-coordinate transformation \((A, B) \to (R, \theta)\), where \(R = \sqrt{A^2 + B^2}\) and \(\tan\theta = B/A\). It can be verified using the cosine angle-addition formula \(\cos(a - b) = \cos a\cos b + \sin a\sin b\).

4. Undamped Motion (\(c = 0\))

When \(c = 0\), the solution is periodic. Define the natural (angular/circular) frequency:

\[ \omega_0 = \sqrt{\frac{k}{m}}. \]

The general solution is

\[ x(t) = c_1\cos(\omega_0 t) + c_2\sin(\omega_0 t) = C\cos(\omega_0 t - \alpha), \]

where \(C = \sqrt{c_1^2 + c_2^2}\) is the amplitude and \(\tan\alpha = c_2/c_1\) gives the phase angle.

The period is \(T = \dfrac{2\pi}{\omega_0}\). Due to the lack of damping, the spring oscillates forever with constant amplitude \(C\).

Example

Consider \(4x'' + 8x = 0,\quad x(0) = 4,\quad x'(0) = 4.\)

Solve for \(x(t)\) and determine the amplitude, period, and all positive times the spring is most compressed.

Solution

Here \(m = 4\), \(c = 0\), \(k = 8\) (undamped). The characteristic equation is \(4r^2 + 8 = 0\), so \(r^2 = -2\), giving \(r = \pm\sqrt{2}\,i\).

General solution:

\[ x(t) = c_1\cos(\sqrt{2}\,t) + c_2\sin(\sqrt{2}\,t). \]

Derivative: \(x'(t) = -\sqrt{2}\,c_1\sin(\sqrt{2}\,t) + \sqrt{2}\,c_2\cos(\sqrt{2}\,t).\)

Applying initial conditions:

\[ x(0) = c_1 = 4, \qquad x'(0) = \sqrt{2}\,c_2 = 4 \;\Longrightarrow\; c_2 = \frac{4}{\sqrt{2}} = 2\sqrt{2}. \]

Therefore:

\[ x(t) = 4\cos(\sqrt{2}\,t) + 2\sqrt{2}\sin(\sqrt{2}\,t). \]

Converting \((A, B) = (4,\; 2\sqrt{2})\) to polar coordinates:

\[ R = \sqrt{4^2 + (2\sqrt{2})^2} = \sqrt{16 + 8} = \sqrt{24}, \qquad \theta = \arctan\!\left(\frac{2\sqrt{2}}{4}\right) = \arctan\!\left(\frac{1}{\sqrt{2}}\right). \]

Since \((4, 2\sqrt{2})\) is in the first quadrant, \(\alpha = \arctan(1/\sqrt{2})\). So:

\[ x(t) = \sqrt{24}\;\cos\!\left(\sqrt{2}\,t - \arctan\!\left(\tfrac{1}{\sqrt{2}}\right)\right). \]

Results:

  • Amplitude: \(\sqrt{24} = 2\sqrt{6}\)
  • Angular frequency: \(\omega_0 = \sqrt{2}\) rad/sec; Hz frequency \(f = \dfrac{\sqrt{2}}{2\pi}\) cycles/sec
  • Period: \(T = \dfrac{2\pi}{\sqrt{2}} = \sqrt{2}\,\pi\) sec

The spring is most compressed when \(x(t)\) reaches its minimum, i.e., when \(\cos(\cdots) = -1\). This requires

\[ \sqrt{2}\,t - \arctan\!\left(\tfrac{1}{\sqrt{2}}\right) = (2n+1)\pi, \qquad n \ge 0 \text{ integer}. \]

Solving for \(t\):

\[ t = \frac{(2n+1)\pi + \arctan\!\left(\frac{1}{\sqrt{2}}\right)}{\sqrt{2}}, \qquad n = 0, 1, 2, \ldots \]

5. Underdamped Motion (\(0 < c < 2\sqrt{km}\))

There is damping, but not enough to prevent oscillations. The complex roots are

\[ r_{1,2} = -\frac{c}{2m} \pm \omega_1 i, \]

where the pseudofrequency (angular) is

\[ \omega_1 = \frac{\sqrt{4km - c^2}}{2m} = \sqrt{\frac{k}{m} - \frac{c^2}{4m^2}}. \]

Note that \(\omega_1 < \omega_0 = \sqrt{k/m}\): damping lowers the frequency of oscillation.

The general solution is

\[ x(t) = c_1 e^{-(c/2m)t}\cos(\omega_1 t) + c_2 e^{-(c/2m)t}\sin(\omega_1 t) = C e^{-(c/2m)t}\cos(\omega_1 t - \alpha), \]

where \(C = \sqrt{c_1^2 + c_2^2}\) and \(\tan\alpha = c_2/c_1\).

Key properties:

  • The time-varying amplitude is \(Ce^{-(c/2m)t}\), which decays exponentially.
  • The motion is not periodic: \(\displaystyle\lim_{t\to\infty} x(t) = 0\) and \(\displaystyle\lim_{t\to\infty} x'(t) = 0\). Oscillations "die out" but never completely stop.
  • The zero crossings (\(x = 0\)) are evenly spaced, separated by \(\dfrac{\pi}{\omega_1}\).
  • We call \(\omega_1\) the pseudofrequency and \(\dfrac{2\pi}{\omega_1}\) the pseudoperiod.
Example

Consider \(x'' + 2x' + 101x = 0,\quad x(0) = 1,\quad x'(0) = -5.\)

Find \(x(t)\), the time-varying amplitude, pseudofrequency, pseudoperiod, and all positive times with a local extremum.

Solution

Here \(m = 1\), \(c = 2\), \(k = 101\). Characteristic equation: \(r^2 + 2r + 101 = 0\), i.e., \((r+1)^2 + 100 = 0\), giving \(r = -1 \pm 10i\). This is underdamped.

General solution:

\[ x(t) = c_1 e^{-t}\cos(10t) + c_2 e^{-t}\sin(10t). \]

Derivative:

\[ x'(t) = -c_1 e^{-t}\cos(10t) - 10c_1 e^{-t}\sin(10t) - c_2 e^{-t}\sin(10t) + 10c_2 e^{-t}\cos(10t). \]

From initial conditions: \(c_1 = 1\), and \(-1 + 10c_2 = -5\) gives \(c_2 = -\tfrac{2}{5}\). So:

\[ x(t) = e^{-t}\!\left(\cos(10t) - \tfrac{2}{5}\sin(10t)\right). \]

Converting \((X, Y) = (1, -\tfrac{2}{5})\) to polar: fourth quadrant, so

\[ R = \sqrt{1 + \tfrac{4}{25}} = \frac{\sqrt{29}}{5}, \qquad \theta = \arctan\!\left(-\tfrac{2}{5}\right) = -\arctan\!\left(\tfrac{2}{5}\right). \]

Therefore:

\[ \boxed{x(t) = \frac{\sqrt{29}}{5}\,e^{-t}\cos\!\left(10t + \arctan\!\left(\tfrac{2}{5}\right)\right).} \]
  • Time-varying amplitude: \(\dfrac{\sqrt{29}}{5}\,e^{-t}\)
  • Pseudofrequency: \(\omega_1 = 10\) rad/sec
  • Pseudoperiod: \(\dfrac{2\pi}{10} = \dfrac{\pi}{5}\) sec

Finding local extrema

Set \(x'(t) = 0\). Differentiating the amplitude-phase form and dividing out the (nonzero) factors \(\dfrac{\sqrt{29}}{5}\,e^{-t}\), we need:

\[ -\cos\!\left(10t + \arctan\!\tfrac{2}{5}\right) - 10\sin\!\left(10t + \arctan\!\tfrac{2}{5}\right) = 0. \]

This gives

\[ \tan\!\left(10t + \arctan\!\tfrac{2}{5}\right) = -\frac{1}{10}. \]

Since tangent is \(\pi\)-periodic:

\[ 10t + \arctan\!\left(\tfrac{2}{5}\right) = \arctan\!\left(-\tfrac{1}{10}\right) + n\pi, \qquad n \ge 0 \text{ integer}. \]

Solving for \(t\):

\[ t = \frac{-\arctan\!\left(\frac{1}{10}\right) - \arctan\!\left(\frac{2}{5}\right) + n\pi}{10}, \qquad n \ge 1 \text{ (integer, chosen so } t > 0\text{)}. \]

6. Critically Damped (\(c = 2\sqrt{km}\))

The characteristic equation has a repeated negative real root \(r = -\dfrac{c}{2m}\). The general solution is

\[ x(t) = c_1 e^{rt} + c_2\,t\,e^{rt} = e^{-(c/2m)t}(c_1 + c_2 t). \]

Since the roots are real, no cosines or sines appear, and there are no oscillations. The mass returns to equilibrium without oscillating, and \(x(t) \to 0\) as \(t \to \infty\).

The critically damped case (almost) always has exactly one time \(T\) where \(x(T) = 0\), namely \(T = -c_1/c_2\) (provided \(c_2 \neq 0\)). Depending on initial conditions, \(T\) may be positive or negative.

Example

Consider \(x'' + 8x' + 16x = 0,\quad x(0) = 5,\quad x'(0) = -10.\)

Determine all positive times at which the mass passes through equilibrium.

Solution

Here \(m = 1\), \(c = 8\), \(k = 16\). Characteristic equation: \(r^2 + 8r + 16 = (r+4)^2 = 0\), so \(r = -4\) (double root). This is critically damped.

\[ x(t) = c_1 e^{-4t} + c_2\,t\,e^{-4t}. \]

Derivative: \(x'(t) = -4c_1 e^{-4t} + c_2 e^{-4t} - 4c_2 t\,e^{-4t}.\)

From \(x(0) = 5\): \(c_1 = 5\). From \(x'(0) = -10\): \(-20 + c_2 = -10\), so \(c_2 = 10\).

\[ x(t) = 5e^{-4t} + 10t\,e^{-4t} = e^{-4t}(5 + 10t). \]

Setting \(x(t) = 0\): since \(e^{-4t} > 0\) always, we need \(5 + 10t = 0\), giving \(t = -\frac{1}{2}\).

Since \(t = -\frac{1}{2} < 0\), there are no positive times at which the mass passes through equilibrium.

7. Overdamped (\(c > 2\sqrt{km}\))

Since \(c^2 - 4km > 0\), the roots are two distinct negative reals \(r_1, r_2\) with \(r_1 \neq r_2\). The general solution is

\[ x(t) = c_1 e^{r_1 t} + c_2 e^{r_2 t}. \]

Again, no cosines/sines appear, so there are no oscillations. The mass may pass through equilibrium at most once (or not at all), depending on initial conditions. In either case \(x(t) \to 0\) as \(t \to \infty\).

Example

Consider \(\tfrac{1}{2}x'' + 3x' + 4x = 0,\quad x(0) = 2,\quad x'(0) = 0.\)

Determine all positive times at which the mass passes through equilibrium.

Solution

Here \(m = \frac{1}{2}\), \(c = 3\), \(k = 4\). Multiplying through by 2: \(r^2 + 6r + 8 = (r+2)(r+4) = 0\), so \(r = -2, -4\). This is overdamped.

\[ x(t) = c_1 e^{-2t} + c_2 e^{-4t}. \]

Derivative: \(x'(t) = -2c_1 e^{-2t} - 4c_2 e^{-4t}.\)

From \(x(0) = 2\): \(c_1 + c_2 = 2\). From \(x'(0) = 0\): \(-2c_1 - 4c_2 = 0\), so \(c_1 = -2c_2\).

Solving: \(c_2 = -2\), \(c_1 = 4\).

\[ x(t) = 4e^{-2t} - 2e^{-4t}. \]

Setting \(x(t) = 0\): \(4e^{-2t} = 2e^{-4t}\). Multiplying both sides by \(e^{4t}\):

\[ 4e^{2t} = 2 \;\Longrightarrow\; e^{2t} = \tfrac{1}{2} \;\Longrightarrow\; t = \tfrac{1}{2}\ln\!\left(\tfrac{1}{2}\right) < 0. \]

Since \(t < 0\), there are no positive times at which the mass passes through equilibrium.

8. Summary of Behavior

Undamped and underdamped systems pass through the equilibrium position \(x = 0\) infinitely many times. For the undamped case the behavior repeats exactly and is truly periodic; for the underdamped case there is a pseudoperiod related to the spacing of zero crossings, but the amplitude decays exponentially.

Critically damped systems generically have exactly one time \(T\) with \(x(T) = 0\) (namely \(T = -c_1/c_2\)), which may be positive or negative depending on initial conditions.

Overdamped systems may have at most one zero crossing, and for some initial conditions will have none at all.

Example — Determining oscillation from \(c\)

Consider \(x'' + cx' + 5x = 0,\quad x(0) = 1,\quad x'(0) = 0.\)

  1. For what values of \(c\) will the mass oscillate and pass through equilibrium infinitely many times?
    We need the undamped or underdamped case: \(c^2 - 4km < 0\), i.e., \(c^2 - 20 < 0\).
    \[ \boxed{0 \le c < \sqrt{20} = 2\sqrt{5}.} \]
  2. If \(c = 6\), can any choice of \(x'(0)\) produce infinite oscillations?
    No. With \(c = 6 > 2\sqrt{5}\), the system is overdamped regardless of initial conditions. The characteristic roots are \(r = \frac{-6 \pm \sqrt{36 - 20}}{2} = \frac{-6 \pm 4}{2} = -1, -5\), both real and negative. Changing \(x'(0)\) changes the coefficients \(c_1, c_2\) but cannot change the nature of the roots.

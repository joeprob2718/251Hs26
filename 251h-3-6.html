1. Undamped Forced Oscillations

We consider the inhomogeneous mass–spring system:

\[ mx'' + cx' + kx = F_0 \cos(\omega t) \]

Here \(F_0\) is the driving force amplitude and \(\omega\) is the driving force frequency.

In the undamped case (\(c = 0\)), we need to solve

\[ mx'' + kx = F_0 \cos(\omega t). \]

In Section 3.5 we discussed how to use undetermined coefficients to handle this. There are two cases depending on whether the driving frequency \(\omega\) matches the natural frequency \(\omega_0 = \sqrt{k/m}\).

Case 1: \(\omega \neq \omega_0 = \sqrt{k/m}\)

There is no matchup between the inhomogeneity and \(x_c\), so the general solution is

\[ x = x_c + x_p = c_1 \cos\!\Bigl(\sqrt{\tfrac{k}{m}}\,t\Bigr) + c_2 \sin\!\Bigl(\sqrt{\tfrac{k}{m}}\,t\Bigr) + A\cos(\omega t) + B\sin(\omega t) \]

which can be rewritten as

\[ x = C\cos\!\Bigl(\sqrt{\tfrac{k}{m}}\,t - \alpha\Bigr) + D\cos(\omega t - \beta). \]

The solution is periodic (provided the frequency ratio \(\omega/\omega_0\) is rational; otherwise it is quasiperiodic) and bounded since

\[ -(C + D) \le x(t) \le C + D \quad \text{for all } t. \]

Case 2: \(\omega = \omega_0 = \sqrt{k/m}\)

In this case the driving frequency matches the natural frequency, so we need to adjust our \(x_p\) ansatz. This leads to

\[ x = x_c + x_p = c_1\cos(\omega t) + c_2\sin(\omega t) + At\cos(\omega t) + Bt\sin(\omega t) \]

or equivalently

\[ x = C\cos(\omega t - \alpha) + Dt\cos(\omega t - \beta). \]

The time-varying amplitude of the second term is \(D_t = \sqrt{A^2 + B^2}\,t\), which satisfies \(\displaystyle\lim_{t \to \infty} D_t = \infty\). This contrasts with the underdamped case, where the amplitude decayed exponentially.

Key takeaway: When an undamped mass–spring system is driven at its natural frequency, the time-varying amplitude grows linearly without bound.

Because no physical system can sustain arbitrarily large oscillations, it would break at some finite time — the classic example is a wine glass shattering when an opera singer hits the "exactly right" note!

2. Resonance

We consider the forced undamped spring system:

\[ mx'' + kx = F_0\cos(\omega t). \]

"Exact Undamped Resonance"

If the driving frequency \(\omega\) coincides with the natural frequency \(\omega_0 = \sqrt{k/m}\), the particular solution

\[ x_p = At\cos(\omega_0 t) + Bt\sin(\omega_0 t) = Ct\cos(\omega_0 t - \alpha) \]

has a linearly-growing time-varying amplitude.

"Approximate Undamped Resonance"

Even if the driving frequency \(\omega\) isn't exactly equal to \(\omega_0\), for \(\omega \approx \omega_0\) the amplitude \(|C|\) in

\[ x_p = A\cos(\omega t) + B\sin(\omega t) = C\cos(\omega t - \alpha) \]

satisfies

\[ \lim_{\omega \to \omega_0} |C| = \infty. \]

A plot of the particular-solution amplitude \(|C|\) against the driving frequency \(\omega\) shows a vertical asymptote at \(\omega = \omega_0\): the amplitude blows up as \(\omega \to \omega_0\).

3. Worked Example

Example

Consider the driven mass–spring system

\[ x'' + 4x = 3\cos(\omega t), \qquad x(0) = x'(0) = 0, \]

where the driving frequency \(\omega\) is unspecified.

  1. Find \(x(t)\) for the two cases \(\omega \neq 2\) and \(\omega = 2\).
  2. What happens to the amplitude of \(x_p\) as \(\omega \to 2\)?

Finding \(x_c\) (same for all \(\omega\))

The complementary equation \(x'' + 4x = 0\) has characteristic equation \(r^2 + 4 = 0\), giving \(r = \pm 2i\). Thus

\[ x_c = c_1\cos(2t) + c_2\sin(2t). \]

Case 1: \(\omega \neq 2\)

The initial guess \(x_p = A\cos(\omega t) + B\sin(\omega t)\) does not match \(x_c\), so no adjustment is needed. Computing derivatives:

\[ x_p' = -A\omega\sin(\omega t) + B\omega\cos(\omega t), \] \[ x_p'' = -A\omega^2\cos(\omega t) - B\omega^2\sin(\omega t). \]

Substituting into \(x'' + 4x = 3\cos(\omega t)\):

\[ (-A\omega^2 + 4A)\cos(\omega t) + (-B\omega^2 + 4B)\sin(\omega t) = 3\cos(\omega t) + 0\sin(\omega t). \]

Matching coefficients (valid since \(\omega \neq 2\)):

\[ A(4 - \omega^2) = 3 \;\Longrightarrow\; A = \frac{3}{4 - \omega^2}, \qquad B(4 - \omega^2) = 0 \;\Longrightarrow\; B = 0. \]

So \(x_p = \dfrac{3}{4 - \omega^2}\cos(\omega t)\) and the general solution is

\[ x = c_1\cos(2t) + c_2\sin(2t) + \frac{3}{4 - \omega^2}\cos(\omega t). \]

Applying initial conditions \(x(0) = 0\) and \(x'(0) = 0\):

\[ c_1 + \frac{3}{4 - \omega^2} = 0 \;\Longrightarrow\; c_1 = -\frac{3}{4 - \omega^2}, \qquad c_2 = 0. \]

Therefore:

\[ \boxed{\;x(t) = \frac{3}{4 - \omega^2}\bigl(\cos(\omega t) - \cos(2t)\bigr)\;} \]

Case 2: \(\omega = 2\)

Now the initial guess \(x_p = A\cos(2t) + B\sin(2t)\) matches \(x_c\), so we must adjust:

\[ x_p = At\cos(2t) + Bt\sin(2t). \]

Computing derivatives (product rule):

\[ x_p' = A\cos(2t) + B\sin(2t) - 2At\sin(2t) + 2Bt\cos(2t), \] \[ x_p'' = -4A\sin(2t) + 4B\cos(2t) - 4At\cos(2t) - 4Bt\sin(2t). \]

Substituting into \(x'' + 4x = 3\cos(2t)\) and simplifying (the \(t\cos(2t)\) and \(t\sin(2t)\) terms cancel):

\[ -4A\sin(2t) + 4B\cos(2t) = 0\cdot\sin(2t) + 3\cos(2t). \]

Matching coefficients:

\[ -4A = 0 \;\Longrightarrow\; A = 0, \qquad 4B = 3 \;\Longrightarrow\; B = \tfrac{3}{4}. \]

So \(x_p = \tfrac{3}{4}\,t\sin(2t)\). Applying initial conditions gives \(c_1 = c_2 = 0\), hence:

\[ \boxed{\;x(t) = \frac{3}{4}\,t\sin(2t)\;} \]

Part 2: Amplitude as \(\omega \to 2\)

From Case 1, the amplitude of \(x_p\) is \(\left|\dfrac{3}{4 - \omega^2}\right|\). As \(\omega \to 2\):

\[ \lim_{\omega \to 2}\left|\frac{3}{4 - \omega^2}\right| = +\infty. \]

4. Practical Resonance: Damped Forced Oscillations

All real-world scenarios involve friction, so we should consider

\[ mx'' + cx' + kx = F_0\cos(\omega t) \]

where \(c > 0\).

For \(c > 0\), "true, exact resonance" cannot occur — there won't be a matchup between \(x_c\) and \(x_p\), and we don't have \(\displaystyle\lim_{\omega \to \omega_m} |C(\omega)| = \infty\) for any frequency \(\omega_m\).

However, for \(0 < c < \sqrt{2km} = \dfrac{c_{cr}}{\sqrt{2}}\) (roughly 70% of critical damping), there is a global maximum of \(|C(\omega)|\) at a certain frequency \(\omega_m\). This "finite but possibly large" amplitude could still cause problems if it exceeds what the system can sustain.

Summary of amplitude vs. driving frequency behavior

  • When \(c < \sqrt{2km}\): There exists a frequency \(\omega_{\max}\) at which \(|C(\omega)|\) attains an absolute maximum. The peak can be large enough to be "dangerous."
  • When \(c \ge \sqrt{2km}\): The function \(C'(\omega) < 0\) for all \(\omega > 0\), so the amplitude is strictly decreasing — no resonance peak.

5. Beats

From Math 141, the product-to-sum formula states:

\[ \sin a\,\sin b = \frac{1}{2}\bigl[\cos(a - b) - \cos(a + b)\bigr]. \]

Rearranging with \(a = \dfrac{A+B}{2}\) and \(b = \dfrac{A-B}{2}\) yields the difference-to-product formula:

\[ \cos B - \cos A = 2\sin\!\left(\frac{A - B}{2}\right)\sin\!\left(\frac{A + B}{2}\right). \]

Applying to the IVP

The solution to

\[ mx'' + kx = F_0\cos(\omega t), \qquad x(0) = x'(0) = 0 \]

is

\[ x(t) = \frac{F_0}{m(\omega_0^2 - \omega^2)}\bigl(\cos(\omega t) - \cos(\omega_0 t)\bigr), \]

where \(\omega_0 = \sqrt{k/m}\). Using the difference-to-product formula:

\[ x(t) = \frac{2F_0}{m(\omega_0^2 - \omega^2)} \sin\!\left(\frac{\omega - \omega_0}{2}\,t\right) \sin\!\left(\frac{\omega + \omega_0}{2}\,t\right). \]

If \(\omega \approx \omega_0\), then:

  • The average \(\dfrac{\omega + \omega_0}{2} \approx \omega_0\) — a "fast" oscillation.
  • The half-difference \(\dfrac{\omega - \omega_0}{2} \approx 0\) — a "slow" oscillation.

We interpret \(\displaystyle\frac{2F_0}{m(\omega_0^2 - \omega^2)}\sin\!\left(\frac{\omega - \omega_0}{2}\,t\right)\) as a slowly-varying amplitude modulating the fast oscillation \(\sin\!\left(\frac{\omega + \omega_0}{2}\,t\right)\). This phenomenon produces beats.

6. Application: Piano Tuning and the 12-Tone Scale

Professional piano tuners use these trig identities in practice. Using the standard 12-tone scale, an "equally tempered" instrument averages out the error from the fact that the circle of fifths doesn't perfectly close:

\[ \frac{3^{12}}{2^{18}} \approx 2.0273 \neq 2. \]

Equal temperament ensures each key signature sounds equally "in tune." Instead of tuning perfect fifths, the tuner introduces a small error and listens for the beats of nearby frequencies: a 1 Hz beat arises from a frequency difference of 1 Hz. The trig identity is what lets us quantify how close two frequencies are by counting beats.

The special frequency ratios of 2 (octaves) and 3/2 (perfect fifths) are ubiquitous across cultures. The fact that \(3^{12}/2^{18} \approx 2\) is the reason our scale has 12 tones. The deeper explanation comes from solving the wave equation with Fourier series.

7. Product-to-Sum from Euler's Formula

Recall that Euler's formula, \(e^{ix} = \cos x + i\sin x\), follows from the Maclaurin series. We have

\[ e^{(a+b)i} = \cos(a+b) + i\sin(a+b), \]

and also

\[ e^{(a+b)i} = e^{ai}\,e^{bi} = (\cos a + i\sin a)(\cos b + i\sin b) = (\cos a\cos b - \sin a\sin b) + i(\sin a\cos b + \sin b\cos a). \]

Since two complex numbers in standard form are equal only when both their real and imaginary parts agree:

\[ a + bi = c + di \;\;\text{with } a,b,c,d \text{ real} \;\;\Longrightarrow\;\; \begin{cases} a = c \\ b = d \end{cases} \]

we obtain the angle addition formulas:

\[ \begin{cases} \cos(a+b) = \cos a\cos b - \sin a\sin b \\ \sin(a+b) = \sin a\cos b + \sin b\cos a \end{cases} \]

Using the fact that cosine is even (\(\cos(-b) = \cos b\)) and sine is odd (\(\sin(-b) = -\sin b\)), you can quickly derive the angle subtraction formulas as well.

† The matching of real and imaginary parts works because the complex numbers form a 2-dimensional real vector space with basis \(\{1, i\}\). Writing \(z = c_1 \cdot 1 + c_2 \cdot i\) is just like working with vector components, which is why it feels similar to the "linearly independent function coefficient matching" used throughout this section.

8. de Moivre's Formula

de Moivre's formula gives trig identities for \(\cos(n\theta)\) and \(\sin(n\theta)\) via Euler's formula. These reveal linear dependence relations useful in undetermined coefficients computations.

Case \(n = 2\)

On one hand, \(e^{2i\theta} = \cos(2\theta) + i\sin(2\theta)\). On the other hand:

\[ e^{2i\theta} = (e^{i\theta})^2 = (\cos\theta + i\sin\theta)^2 = (\cos^2\theta - \sin^2\theta) + i(2\sin\theta\cos\theta). \]

Therefore:

\[ \begin{cases} \cos(2\theta) = \cos^2\theta - \sin^2\theta \\ \sin(2\theta) = 2\sin\theta\cos\theta \end{cases} \]

Using \(\sin^2\theta + \cos^2\theta = 1\) with the \(\cos(2\theta)\) identity gives the power-reduction formulas:

\[ \cos^2\theta = \frac{1}{2} + \frac{1}{2}\cos(2\theta), \qquad \sin^2\theta = \frac{1}{2} - \frac{1}{2}\cos(2\theta). \]

Case \(n = 3\)

Similarly, \(e^{3i\theta} = \cos(3\theta) + i\sin(3\theta)\), and:

\[ e^{3i\theta} = (e^{i\theta})^3 = (\cos\theta + i\sin\theta)^3 = (\cos^3\theta - 3\cos\theta\sin^2\theta) + i(3\cos^2\theta\sin\theta - \sin^3\theta). \]

Therefore:

\[ \begin{cases} \cos(3\theta) = \cos^3\theta - 3\cos\theta\sin^2\theta = 4\cos^3\theta - 3\cos\theta \\[6pt] \sin(3\theta) = 3\cos^2\theta\sin\theta - \sin^3\theta = 3\sin\theta - 4\sin^3\theta \end{cases} \]

Rearranging gives identities for \(\cos^3\theta\) and \(\sin^3\theta\). For instance:

\[ \cos^3\theta = \frac{1}{4}\cos(3\theta) + \frac{3}{4}\cos\theta. \]

Section 1.4: Separable First-Order DEs

Annotated Lecture Notes

Introduction

Unfortunately, even for first-order ODEs, we won't always be able to make progress solving them. They'll need to be of special types. In this course we'll discuss two solvable types:

  • First-order separable (Section 1.4 — this lecture)
  • First-order linear (Section 1.5)

Separable DEs

Recall the general first-order DE

\[\frac{dy}{dx} = f(x, y).\]

For today, we consider the case where \(f(x,y)\) has the special form \(g(x)\,k(y)\), so that the DE is of the form

\[\frac{dy}{dx} = g(x)\,k(y).\]

This is called a separable DE because we can separate the \(x\)-stuff from the \(y\)-stuff.

Special case — autonomous ODEs: An autonomous ODE \(\dfrac{dy}{dx} = k(y)\) is separable with \(g(x) = 1\) (a constant function). So autonomous equations are a special case of separable equations.
Not every first-order ODE is separable! For example, \(\dfrac{dy}{dx} = x + y\) is not of the form \(g(x)\,k(y)\) — it cannot be factored as a product of a function of \(x\) alone and a function of \(y\) alone. (This one is first-order linear, which we can solve in Section 1.5.) Similarly, \(\dfrac{dy}{dx} = x^2 + y^2\) is neither separable nor linear — we won't be able to solve it with these techniques.

Solving Separable DEs

To solve a separable DE, we "abuse notation" and treat \(\dfrac{dy}{dx}\) as a fraction. Starting from

\[\frac{dy}{dx} = g(x)\,k(y),\]

we multiply both sides by \(dx\) and divide both sides by \(k(y)\) to obtain

\[\frac{1}{k(y)}\, dy = g(x)\, dx,\]

and then integrate both sides:

\[\int \frac{1}{k(y)}\, dy = \int g(x)\, dx.\]

We then proceed with typical antiderivatives, and eventually solve for \(y\).

Sometimes the algebra steps to isolate \(y\) are difficult or impossible, so we may occasionally only ask for implicit solutions, where you stop right after the antiderivatives. An implicit solution describes a relationship between \(x\) and \(y\) without explicitly expressing \(y\) as a function of \(x\).

Example of an implicit solution you can't make explicit: If you arrive at \(x = \ln(2 - x)\), you cannot solve for \(x\) algebraically. Sometimes we have to live with implicit solutions.

Justification of the "Fraction" Notation

The notation trick of treating \(\dfrac{dy}{dx}\) as a fraction is not actually rigorous — \(\dfrac{dy}{dx}\) is not literally a fraction! (It's the limit \(\displaystyle\lim_{\Delta x \to 0}\frac{\Delta y}{\Delta x}\).) But the trick gives the right answer. Here's why:

Rewrite \(\dfrac{dy}{dx} = g(x)\,k(y)\) as

\[\frac{1}{k(y)}\,\frac{dy}{dx} = g(x).\]

Integrate both sides with respect to \(x\), recalling that \(y = y(x)\) so that \(y'(x) = \dfrac{dy}{dx}\):

\[\int \frac{1}{k\!\big(y(x)\big)}\,y'(x)\, dx = \int g(x)\, dx.\]

Now implement the substitution \(y = y(x)\), \(dy = y'(x)\,dx\) on the left side:

\[\int \frac{1}{k(y)}\, dy = \int g(x)\, dx.\]

"Treating \(\dfrac{dy}{dx}\) as a fraction" to help us solve separable ODEs is the same mnemonic device we use for setting up the differentials when integrating by substitution!

Chain rule mnemonic: This is exactly like the chain rule shorthand: if \(z\) depends on \(y\) and \(y\) depends on \(x\), then \(\dfrac{dz}{dx} = \dfrac{dz}{dy} \cdot \dfrac{dy}{dx}\), which "looks like" canceling fractions.

Example: Exponential Growth/Decay

Example

Find all solutions to the differential equation \(\dfrac{dy}{dx} = ky\).

Step 1: Separate variables

Assuming \(y \ne 0\) (we divided by \(y\) — we'll come back to this), we get:

\[dy = ky\,dx \quad \Longrightarrow \quad \frac{1}{y}\,dy = k\,dx\]

Step 2: Integrate both sides

\[\int \frac{1}{y}\,dy = \int k\,dx\]
Important: You must include the constant of integration \(+C\) at the antidifferentiation step! It is enough to put \(+C\) on one side only. Adding it at the end or guessing where it should go will very likely give the wrong family of functions.
\[\ln|y| = kx + C_1\]

This is the implicit solution — it describes a relationship between \(x\) and \(y\).

Step 3: Solve for \(y\) explicitly

Exponentiate both sides:

\[e^{\ln|y|} = e^{kx + C_1}\] \[|y| = e^{kx} \cdot e^{C_1} = C_2\, e^{kx}\]

where \(C_2 = e^{C_1}\) is any positive constant. Dropping the absolute value:

\[y = \pm C_2\, e^{kx} = C\,e^{kx}\]

where \(C\) is any nonzero constant.

Step 4: Check for singular solutions

We assumed \(y \ne 0\) when we divided by \(y\). Does \(y = 0\) work? Check: if \(y = 0\) for all \(x\), then \(\dfrac{dy}{dx} = 0\) and \(ky = k \cdot 0 = 0\). ✓ Yes!

Notice that \(C = 0\) gives \(y = 0\), so the constant solution is actually included in the formula.

Full list of solutions: \(y = Ce^{kx}\), where \(C\) is any constant.

Useful antiderivative reminder: \(\displaystyle\int y^p\,dy = \begin{cases} \ln|y| & \text{if } p = -1 \\[6pt] \dfrac{y^{p+1}}{p+1} & \text{if } p \ne -1 \end{cases}\)

Example: \(y' = ky^2\)

Example

Find all solutions to the differential equation \(\dfrac{dy}{dx} = ky^2\).

Step 1: Separate variables

Assuming \(y \ne 0\):

\[dy = ky^2\,dx \quad \Longrightarrow \quad \frac{1}{y^2}\,dy = k\,dx\]

Step 2: Integrate both sides

\[\int y^{-2}\,dy = \int k\,dx\] \[\frac{y^{-1}}{-1} = kx + C\] \[-\frac{1}{y} = kx + C\]

Step 3: Solve for \(y\)

\[y = -\frac{1}{kx + C}\]

where \(C\) is any constant.

Step 4: Check for singular solutions

We assumed \(y \ne 0\). Check: \(y = 0\) gives \(\dfrac{dy}{dx} = 0\) and \(k \cdot 0^2 = 0\). ✓

Full list of solutions: \(y = -\dfrac{1}{kx + C}\) for any constant \(C\), and \(y = 0\) (constant solution).

Common mistake: Do NOT try to split \(-\dfrac{1}{y}\) into separate fractions like \(-\dfrac{1}{kx} - \dfrac{1}{C}\). The expression \(\dfrac{1}{a+b} \ne \dfrac{1}{a} + \dfrac{1}{b}\)! Keep the denominator as a single expression \(kx + C\).

Singular/Exceptional Solutions

It is possible for the general solution to miss finitely many solutions — these are called singular or exceptional solutions.

You'll know which "missing" solutions to expect by looking through your steps and checking to see anything you divided by — as soon as you divide by something, you are assuming it to be nonzero!

For a separable equation \(\dfrac{dy}{dx} = g(x)\,k(y)\), the only missing solutions will be constant solutions — values \(y = y_0\) where \(k(y_0) = 0\). These are also called equilibrium solutions or fixed points (we'll study these more in Chapter 2).

Procedure: After finding the general solution via separation of variables, always go back and check whether any constant solutions \(y = y_0\) (where \(k(y_0) = 0\)) are included in your formula or need to be listed separately.

The Importance of \(+C\) at the Right Step

It is extremely important to include the additive integration constant at the antidifferentiation step!

Realizing you forgot and doing \(+C\) at the end, or trying to guess how it should appear, will very likely give you the wrong family of functions.

Rule of thumb: Always introduce the constant of integration immediately when you perform the antiderivative, on one side of the equation. Then carry it through all subsequent algebra.

Example: The Logistic Equation

Example

Find all solutions to the logistic differential equation

\[\frac{dP}{dt} = kP(M - P)\]

where \(k > 0\) and \(M\) is the carrying capacity.

Step 1: Separate variables

Assuming \(P \ne 0\) and \(P \ne M\) (we're dividing by \(P(M - P)\)):

\[\frac{1}{P(M - P)}\,dP = k\,dt\]

Step 2: Integrate both sides

\[\int \frac{1}{P(M - P)}\,dP = \int k\,dt = kt + C_1\]

The left-hand integral requires partial fractions.

Partial fraction decomposition

We want to write

\[\frac{1}{P(M - P)} = \frac{A}{P} + \frac{B}{M - P}.\]

Combining the right side over a common denominator:

\[\frac{A(M - P) + BP}{P(M - P)} = \frac{(-A + B)P + AM}{P(M - P)}.\]

For this to equal \(\dfrac{1}{P(M-P)} = \dfrac{0 \cdot P + 1}{P(M-P)}\) for all \(P\), we need the coefficients of each power of \(P\) to match (two linear functions of \(P\) can only be equal if their slopes and intercepts separately match):

\[\begin{cases} -A + B = 0 \\ AM = 1 \end{cases} \quad \Longrightarrow \quad A = \frac{1}{M}, \quad B = \frac{1}{M}.\]

Performing the integration

\[\int \left(\frac{1/M}{P} + \frac{1/M}{M - P}\right) dP = kt + C_1\]

For the second term, use the substitution \(u = M - P\), \(du = -dP\):

\[\frac{1}{M}\ln|P| - \frac{1}{M}\ln|M - P| = kt + C_1\]

Multiply both sides by \(M\):

\[\ln|P| - \ln|M - P| = Mkt + C_2 \qquad (C_2 = MC_1)\] \[\ln\left|\frac{P}{M - P}\right| = Mkt + C_2\]

Solving for \(P\)

Exponentiate both sides:

\[\left|\frac{P}{M - P}\right| = e^{Mkt + C_2} = e^{C_2} \cdot e^{Mkt}\] \[\frac{P}{M - P} = \pm e^{C_2} \cdot e^{Mkt} = C_3\, e^{Mkt}\]

where \(C_3 = \pm e^{C_2}\) is any nonzero constant. Solving for \(P\):

\[P = (M - P)\,C_3\,e^{Mkt} = MC_3\,e^{Mkt} - PC_3\,e^{Mkt}\] \[P + PC_3\,e^{Mkt} = MC_3\,e^{Mkt}\] \[P\!\left(1 + C_3\,e^{Mkt}\right) = MC_3\,e^{Mkt}\] \[P = \frac{MC_3\,e^{Mkt}}{1 + C_3\,e^{Mkt}}\]

To simplify, multiply numerator and denominator by \(\dfrac{1}{C_3}\,e^{-Mkt}\), letting \(C = \dfrac{1}{C_3}\):

\[P(t) = \frac{M}{C\,e^{-Mkt} + 1}\]

where \(C\) is any constant.

Singular solutions

We assumed \(P \ne 0\) and \(P \ne M\). Check both:

  • \(P = 0\) for all \(t\): \(\dfrac{dP}{dt} = 0\) and \(kP(M - P) = 0\). ✓
  • \(P = M\) for all \(t\): \(\dfrac{dP}{dt} = 0\) and \(kM(M - M) = 0\). ✓

Both are equilibrium solutions that should be listed if not already captured by the formula.

Looking ahead: In Chapter 2.2, we'll be able to quickly (without calculus!) get qualitative information about solutions to autonomous ODEs like this one. Sometimes the actual solution formulas are quite tedious — qualitative methods can tell us a lot with much less work.

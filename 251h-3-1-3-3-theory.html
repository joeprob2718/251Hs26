3.1, 3.2, 3.3: Theory for Linear DEs of Higher Order, Extending the 2nd Order Techniques

Higher Order Equations

Everything we discussed for the second order case extends to higher order. An \(n\)th order ODE requires \(n\) linearly independent solutions to build the general solution.

Repeated complex roots are possible and treated similarly to repeated real roots. For roots with multiplicity greater than 2, we "keep multiplying by enough \(x\)'s to get enough linearly independent solutions." The Clairaut's theorem argument from the second order case can be adapted to explain why this works.

Fact — Factoring Real Polynomials

Polynomials with real coefficients can always be factored into a product of simple linear, repeated linear, irreducible quadratic, and repeated irreducible quadratic factors. Finding these factors exactly may not be possible for degree 5 or higher (no quintic formula!), but numerical approximations can be done on a computer. Nonreal roots only arise from irreducible quadratic factors, and they always come in conjugate pairs.

Example: A 13th Order Equation

Example

Write the general solution for the equation whose characteristic polynomial is

\[ (r-1)(r-2)(r-3)^2(r-4)^3\bigl[(r-1)^2+1\bigr]\bigl[(r-1)^2+4\bigr]^2 = 0. \]

This is a 13th order constant coefficient linear homogeneous ODE. Let's read off the roots from each factor:

\(r = 1\) (simple root), \(r = 2\) (simple root), \(r = 3\) (double root), \(r = 4\) (triple root).

From \((r-1)^2 + 1 = 0\): \(r = 1 \pm i\) (simple complex pair).

From \([(r-1)^2 + 4]^2 = 0\): \(r = 1 \pm 2i\) (double complex pair).

The general solution is:

\[ \begin{aligned} y &= c_1 e^{x} + c_2 e^{2x} \\ &\quad + c_3 e^{3x} + c_4\, x\,e^{3x} \\ &\quad + c_5 e^{4x} + c_6\, x\,e^{4x} + c_7\, x^2 e^{4x} \\ &\quad + c_8\, e^{x}\cos(x) + c_9\, e^{x}\sin(x) \\ &\quad + c_{10}\, e^{x}\cos(2x) + c_{11}\, e^{x}\sin(2x) \\ &\quad + c_{12}\, x\,e^{x}\cos(2x) + c_{13}\, x\,e^{x}\sin(2x). \end{aligned} \]

Notice the pattern: each root of multiplicity \(m\) contributes \(m\) solutions, obtained by multiplying the base solution by \(1, x, x^2, \ldots, x^{m-1}\). For complex pairs, each power of \(x\) appears with both the cosine and sine terms.

Example: Third Order Equation

Example

Find the general solution of \(\;9y^{(3)} + 12y'' + 4y' = 0\).

Characteristic polynomial: \(9r^3 + 12r^2 + 4r = 0\). Factor out \(r\):

\[ r(9r^2 + 12r + 4) = 0 \quad\Longrightarrow\quad r(3r+2)^2 = 0. \]

Roots: \(r = 0\) (simple) and \(r = -\tfrac{2}{3}\) (double).

Note that \(e^{0 \cdot x} = 1\), so the general solution is:

\[ \boxed{y = c_1 + c_2\, e^{-\frac{2}{3}x} + c_3\, x\, e^{-\frac{2}{3}x}.} \]

Example: Fourth Order — Double Roots

Example

Find the general solution of \(\;y^{(4)} - 8y'' + 16y = 0\).

Characteristic polynomial: \(r^4 - 8r^2 + 16 = 0\). This is a quadratic in \(r^2\):

\[ (r^2 - 4)^2 = 0 \quad\Longrightarrow\quad \bigl[(r-2)(r+2)\bigr]^2 = 0 \quad\Longrightarrow\quad (r-2)^2(r+2)^2 = 0. \]

Roots: \(r = 2\) (double) and \(r = -2\) (double).

\[ \boxed{y = c_1\, e^{2x} + c_2\, x\, e^{2x} + c_3\, e^{-2x} + c_4\, x\, e^{-2x}.} \]

Example: Fourth Order — Triple Root at Zero

Example

Find the general solution of \(\;5y^{(4)} + 4y^{(3)} = 0\).

Characteristic polynomial: \(5r^4 + 4r^3 = 0\). Factor out \(r^3\):

\[ r^3(5r + 4) = 0. \]

Roots: \(r = 0\) (triple) and \(r = -\tfrac{4}{5}\) (simple). Since \(e^{0 \cdot x} = 1\), \(x\,e^{0 \cdot x} = x\), and \(x^2 e^{0 \cdot x} = x^2\):

\[ \boxed{y = c_1 + c_2\, x + c_3\, x^2 + c_4\, e^{-\frac{4}{5}x}.} \]

The General \(n\)th Order Linear DE

An \(n\)th order linear DE has the form

\[ y^{(n)} + p_1(x)\,y^{(n-1)} + \cdots + p_{n-1}(x)\,y' + p_n(x)\,y = f(x), \]

with the homogeneous case being

\[ y^{(n)} + p_1(x)\,y^{(n-1)} + \cdots + p_{n-1}(x)\,y' + p_n(x)\,y = 0. \]

Superposition Principle for \(n\)th Order

Principle of Superposition

If \(y_1, y_2, \ldots, y_n\) are solutions to a homogeneous linear DE, then any linear combination

\[ y = c_1 y_1 + c_2 y_2 + \cdots + c_n y_n \]

(where \(c_1, c_2, \ldots\) are constants) is also a solution.

This works for the same reasons as in the second order case: derivative rules are linear, the equation is linear, and the right-hand side is zero.

Existence and Uniqueness for Linear DEs

Existence and Uniqueness Theorem

Consider the IVP

\[ y^{(n)} + p_1(x)\,y^{(n-1)} + \cdots + p_{n-1}(x)\,y' + p_n(x)\,y = f(x), \]

with initial conditions

\[ y(a) = b_0, \quad y'(a) = b_1, \quad \ldots, \quad y^{(n-1)}(a) = b_{n-1}. \]

Provided \(p_1, p_2, \ldots, p_n\) and \(f\) are continuous on an open interval \(I\) containing \(a\), then there exists a unique solution to the initial value problem on that entire interval \(I\).

This is stronger than the general existence and uniqueness theorem for nonlinear equations. For nonlinear equations, finite-time blowup is possible, so the earlier theorem (from Section 1.3) only guaranteed existence on a possibly small interval. Linear equations are better behaved — the dependence on \(y\) and its derivatives is linear, so partial derivatives with respect to them are well controlled, and solutions exist on the entire interval of continuity of the coefficient functions.

Note: the leading coefficient of \(y^{(n)}\) is written as 1 (we divide through by \(P_0(x)\) if needed). If \(P_0(x) = 0\) at some point, that point is a "singular point" where the order of the equation changes and bad behavior can occur. In the constant coefficient homogeneous case, all coefficients are continuous for all \(x\), so solutions exist for all \(x\).

Linear Dependence and Independence of Functions

Linear Dependence

The \(n\) functions \(f_1, f_2, \ldots, f_n\) are linearly dependent on an interval \(I\) if there exist constants \(c_1, c_2, \ldots, c_n\), not all zero, such that

\[ c_1 f_1(x) + c_2 f_2(x) + \cdots + c_n f_n(x) \equiv 0 \quad \text{for all } x \in I. \]
Linear Independence

The \(n\) functions \(f_1, f_2, \ldots, f_n\) are linearly independent on \(I\) if the only linear combination that is identically zero is the trivial one:

\[ c_1 f_1 + c_2 f_2 + \cdots + c_n f_n \equiv 0 \;\text{ on } I \quad\Longrightarrow\quad c_1 = c_2 = \cdots = c_n = 0. \]

The symbol \(\equiv 0\) means "identically zero" — the function is zero for all \(x\) in \(I\), not just at particular points. For two functions, linear independence simply means they are not constant multiples of each other.

The Wronskian

Recall from Math 220 (linear algebra): to test whether \(n\) vectors are linearly independent, we check whether the determinant of the matrix formed by placing them as columns is nonzero. The Wronskian is the analogous tool for functions.

Wronskian

The Wronskian of \(f_1, f_2, \ldots, f_n\) is the determinant

\[ W(x) = \begin{vmatrix} f_1 & f_2 & \cdots & f_n \\ f_1' & f_2' & \cdots & f_n' \\ \vdots & \vdots & & \vdots \\ f_1^{(n-1)} & f_2^{(n-1)} & \cdots & f_n^{(n-1)} \end{vmatrix}. \]

Each column contains a function and its first \(n-1\) derivatives; all entries are evaluated at \(x\).

Wronskian and Linear Dependence

If \(f_1, \ldots, f_n\) are linearly dependent on \(I\), then one function can be written as a linear combination of the others. By linearity of differentiation, the corresponding column of the Wronskian matrix is a linear combination of the other columns, which forces the determinant to be zero. Therefore:

Linearly dependent on \(I\) \(\;\Longrightarrow\;\) \(W(x) \equiv 0\) on \(I\).

The Wronskian Dichotomy for Solutions

For arbitrary functions, the converse of the above statement is not true in general. However, when the functions are solutions to the same homogeneous linear DE with continuous coefficients, something much stronger holds:

Theorem — Wronskian Dichotomy

Suppose \(y_1, y_2, \ldots, y_n\) are solutions to the homogeneous \(n\)th order linear DE

\[ y^{(n)} + p_1(x)\,y^{(n-1)} + \cdots + p_n(x)\,y = 0 \]

on an open interval \(I\) where each \(p_i\) is continuous. Then there are only two possibilities:

(a) \(y_1, \ldots, y_n\) are linearly dependent and \(W(x) \equiv 0\) on \(I\), OR

(b) \(y_1, \ldots, y_n\) are linearly independent and \(W(x) \neq 0\) for all \(x\) in \(I\).

In other words, for solutions of a linear homogeneous ODE, the Wronskian is either always zero or never zero. There is no in-between. This means you only need to check \(W\) at a single convenient point (e.g. \(x = 0\)) to determine independence.

Why Does This Hold?

Proof Sketch

We already know that linear dependence implies \(W \equiv 0\). For the other direction, suppose \(W(a) = 0\) for some \(a \in I\). Then the system

\[ \begin{cases} c_1 y_1(a) + c_2 y_2(a) + \cdots + c_n y_n(a) = 0 \\ c_1 y_1'(a) + c_2 y_2'(a) + \cdots + c_n y_n'(a) = 0 \\ \qquad\vdots \\ c_1 y_1^{(n-1)}(a) + c_2 y_2^{(n-1)}(a) + \cdots + c_n y_n^{(n-1)}(a) = 0 \end{cases} \]

has a nontrivial solution \(c_1, \ldots, c_n\) (since \(\det = 0\) means the matrix is singular — a standard linear algebra fact). Build the function

\[ Y(x) = c_1 y_1(x) + c_2 y_2(x) + \cdots + c_n y_n(x). \]

By superposition, \(Y(x)\) solves the ODE. By construction, \(Y(a) = Y'(a) = \cdots = Y^{(n-1)}(a) = 0\). But the zero function also solves the ODE with these same initial conditions. By the uniqueness part of the existence and uniqueness theorem, \(Y(x) \equiv 0\) on \(I\). Therefore \(y_1, \ldots, y_n\) are linearly dependent.

General Solution Theorem

Theorem

Suppose \(y_1, y_2, \ldots, y_n\) are linearly independent solutions to the homogeneous \(n\)th order linear DE on an open interval \(I\) (with continuous coefficients). Then

\[ y = c_1 y_1 + c_2 y_2 + \cdots + c_n y_n \]

is the general solution — that is, every solution \(Y(x)\) on \(I\) can be expressed in this form for some choice of constants.

Proof

Let \(Y(x)\) be any solution. Pick any \(a \in I\) and compute \(Y(a) = b_0,\; Y'(a) = b_1,\; \ldots,\; Y^{(n-1)}(a) = b_{n-1}\). Since \(W(a) \neq 0\) (by linear independence), the "Wronskian matrix" is invertible, and we can find unique \(c_1, \ldots, c_n\) such that

\[ Z(x) = c_1 y_1 + c_2 y_2 + \cdots + c_n y_n \]

satisfies the same initial conditions as \(Y(x)\). Both \(Y\) and \(Z\) solve the same ODE with the same initial data. By the existence and uniqueness theorem, \(Y(x) \equiv Z(x)\) on \(I\). Therefore \(Y\) is a linear combination of \(y_1, \ldots, y_n\). \(\square\)

Satisfying Initial Conditions

Given initial data \(y(a) = b_0,\; y'(a) = b_1,\; \ldots,\; y^{(n-1)}(a) = b_{n-1}\), we need to solve

\[ \begin{bmatrix} y_1(a) & y_2(a) & \cdots & y_n(a) \\ y_1'(a) & y_2'(a) & \cdots & y_n'(a) \\ \vdots & \vdots & & \vdots \\ y_1^{(n-1)}(a) & y_2^{(n-1)}(a) & \cdots & y_n^{(n-1)}(a) \end{bmatrix} \begin{bmatrix} c_1 \\ c_2 \\ \vdots \\ c_n \end{bmatrix} = \begin{bmatrix} b_0 \\ b_1 \\ \vdots \\ b_{n-1} \end{bmatrix}. \]

Since \(W(a) \neq 0\), this matrix is invertible and the system has a unique solution. This guarantees we will never encounter an inconsistent system (no solutions) or a redundant system (infinitely many solutions) when matching initial conditions — as long as we built our general solution from linearly independent solutions.

Hyperbolic Trigonometric Functions

Definitions
\[ \sinh(x) := \frac{e^x - e^{-x}}{2}, \qquad \cosh(x) := \frac{e^x + e^{-x}}{2}. \]

Key properties:

\(\sinh(x)\) is odd and \(\cosh(x)\) is even. At zero: \(\sinh(0) = 0\) and \(\cosh(0) = 1\). Their derivatives are \(\frac{d}{dx}\sinh(x) = \cosh(x)\) and \(\frac{d}{dx}\cosh(x) = \sinh(x)\). They satisfy the identity \(\cosh^2(x) - \sinh^2(x) = 1\).

Hyperbolic Trig as an Alternative Basis

Suppose \(k > 0\). There is a nice parallel between two fundamental equations:

For \(y'' + k^2 y = 0\), the roots are \(r = \pm ki\) and the general solution is \(y = c_1\cos(kx) + c_2\sin(kx)\).

For \(y'' - k^2 y = 0\), the roots are \(r = \pm k\) and the general solution can be written as either \(y = d_1 e^{kx} + d_2 e^{-kx}\) or equivalently

\[ y = c_1\cosh(kx) + c_2\sinh(kx). \]

This is a "change of basis" — instead of using \(e^{kx}\) and \(e^{-kx}\) as our two linearly independent solutions, we use the linear combinations \(\cosh(kx)\) and \(\sinh(kx)\). These can be a more convenient basis, particularly when evaluating initial conditions at \(x = 0\), since \(\cosh(0) = 1\), \(\sinh(0) = 0\), \(\cosh'(0) = 0\), and \(\sinh'(0) = 1\).

Summary

Key Takeaways

(1) An \(n\)th order constant coefficient linear homogeneous ODE requires \(n\) linearly independent solutions. Each root of the characteristic polynomial of multiplicity \(m\) contributes \(m\) solutions (multiplied by \(1, x, \ldots, x^{m-1}\)).

(2) The Wronskian provides a test for linear independence of solutions: for solutions of the same homogeneous linear ODE, \(W \neq 0\) at one point implies \(W \neq 0\) everywhere, which implies independence.

(3) The existence and uniqueness theorem for linear DEs guarantees solutions exist on the entire interval of continuity of the coefficients — no finite-time blowup.

(4) The general solution theorem ensures that \(y = c_1 y_1 + \cdots + c_n y_n\) captures all solutions, and initial conditions can always be uniquely satisfied.

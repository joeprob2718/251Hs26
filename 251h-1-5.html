Section 1.5: Linear First Order Equations — Integrating Factors

Recap: General First Order DEs

Recall that a general first order differential equation has the form

\[ \frac{dy}{dx} = f(x, y). \]

In Section 1.4 we considered separable DEs of the form

\[ \frac{dy}{dx} = g(x)\,k(y), \]

which we solved by separating variables and integrating both sides:

\[ \int \frac{1}{k(y)}\,dy = \int g(x)\,dx. \]

First Order Linear DEs

Now we consider equations of the form

\[ \frac{dy}{dx} + P(x)\,y = Q(x). \]

We classify this as first order linear because \(y\) and \(\dfrac{dy}{dx}\) appear linearly — we don't have any quantities like \(y^2\), \(\cos(y)\), \(y \cdot \dfrac{dy}{dx}\), \(\left(\dfrac{dy}{dx}\right)^2\), etc.

However, the coefficient \(P(x)\) and the "source term" \(Q(x)\) are allowed to be nonlinear functions of \(x\). For example,

\[ \frac{dy}{dx} - 2xy = 3x^2 e^{x^2} \]

is a linear first-order DE with \(P(x) = -2x\) and \(Q(x) = 3x^2 e^{x^2}\).

Solving First Order Linear DEs

We multiply both sides of \(\dfrac{dy}{dx} + P(x)\,y = Q(x)\) by the integrating factor

\[ \mu(x) = e^{\int P(x)\,dx}. \]

This gives

\[ \frac{dy}{dx}\,e^{\int P(x)\,dx} + y\,P(x)\,e^{\int P(x)\,dx} = Q(x)\,e^{\int P(x)\,dx}. \]

The integrating factor was cleverly chosen so that the left side is exactly the result of applying the product rule (recall from Calculus 1 that \(\frac{d}{dx}(fg) = f'g + fg'\)):

\[ \frac{d}{dx}\!\left(y\,e^{\int P(x)\,dx}\right) = Q(x)\,e^{\int P(x)\,dx}. \]

We can then simply integrate both sides to "undo" the derivative, and solve for \(y\).

Key Notes
  • There is no need for a \(+C\) inside the integrating factor (when computing \(\int P(x)\,dx\) for \(\mu\)).
  • You do need to include \(+C\) when you integrate both sides to undo the derivative on the left.

Summary of Steps

  1. Write the DE in standard form: \(\dfrac{dy}{dx} + P(x)\,y = Q(x)\).
  2. Compute the integrating factor \(\mu(x) = e^{\int P(x)\,dx}\).
  3. Multiply both sides of the DE by \(\mu(x)\).
  4. Recognize the left side as \(\dfrac{d}{dx}\!\bigl(y\,\mu(x)\bigr)\).
  5. Integrate both sides with respect to \(x\), including \(+C\).
  6. Solve for \(y\).

Example 1

Problem Find the general solution to the differential equation \[ \frac{dy}{dx} - 2xy = 3x^2 e^{x^2}. \]

Solution

The equation is already in standard form \(\dfrac{dy}{dx} + P(x)\,y = Q(x)\) with

\[ P(x) = -2x, \qquad Q(x) = 3x^2 e^{x^2}. \]

Step 1 — Integrating factor:

\[ \mu(x) = e^{\int (-2x)\,dx} = e^{-x^2}. \]

Step 2 — Multiply both sides by \(e^{-x^2}\):

\[ \frac{dy}{dx}\,e^{-x^2} - 2xy\,e^{-x^2} = 3x^2 e^{x^2} \cdot e^{-x^2} = 3x^2. \]

Step 3 — Recognize the product rule on the left:

\[ \frac{d}{dx}\!\left(y\,e^{-x^2}\right) = 3x^2. \]

(Check: by the product rule, \(\frac{d}{dx}(y\,e^{-x^2}) = \frac{dy}{dx}\,e^{-x^2} + y\cdot e^{-x^2}\cdot(-2x)\). ✓)

Step 4 — Integrate both sides:

\[ y\,e^{-x^2} = \int 3x^2\,dx = x^3 + C. \]

Step 5 — Solve for \(y\): multiply both sides by \(e^{x^2}\):

\[ \boxed{y = x^3 e^{x^2} + C\,e^{x^2}.} \]
Note The constant \(C\) cannot be absorbed into \(e^{x^2}\) since \(e^{x^2}\) is not itself a constant.

Example 2

Problem Consider the first order DE \[ (13 + x)\frac{dy}{dx} = 8y. \] Find the general solution two ways: first by treating it as separable, and then by treating it as linear.

Method 1 — Separable

Rewrite as

\[ \frac{dy}{dx} = \frac{8}{13+x}\,y, \]

which has the separable form \(\dfrac{dy}{dx} = g(x)\,k(y)\) with \(g(x)=\dfrac{8}{13+x}\) and \(k(y)=y\). Separate and integrate:

\[ \int \frac{1}{y}\,dy = \int \frac{8}{13+x}\,dx. \] \[ \ln|y| = 8\ln|13+x| + C_1 = \ln\!\left(|13+x|^8\right) + C_1. \]

Exponentiate both sides:

\[ |y| = e^{\ln(|13+x|^8) + C_1} = e^{C_1}\,|13+x|^8. \]

Setting \(C_2 = e^{C_1}\) and noting that 8 is even (so \(|13+x|^8 = (13+x)^8\)):

\[ \boxed{y = C(13+x)^8} \]

where \(C = \pm C_2\) is an arbitrary constant.

Method 2 — First Order Linear (Integrating Factor)

Rewrite the DE in standard form \(\dfrac{dy}{dx} + P(x)\,y = Q(x)\):

\[ \frac{dy}{dx} - \frac{8}{13+x}\,y = 0. \]

So \(P(x) = -\dfrac{8}{13+x}\) and \(Q(x) = 0\).

Integrating factor:

\[ \mu(x) = e^{\int -\frac{8}{13+x}\,dx} = e^{-8\ln|13+x|} = e^{\ln(|13+x|^{-8})} = |13+x|^{-8} = \frac{1}{(13+x)^8}. \]

Multiply the ODE by the integrating factor:

\[ \frac{dy}{dx}\cdot\frac{1}{(13+x)^8} - \frac{8}{(13+x)^9}\,y = 0. \]

Recognize the product rule:

\[ \frac{d}{dx}\!\left(\frac{y}{(13+x)^8}\right) = 0. \]

(Check: \(\frac{d}{dx}\bigl(y\cdot(13+x)^{-8}\bigr) = \frac{dy}{dx}\,(13+x)^{-8} + y\bigl[-8(13+x)^{-9}\bigr]\). ✓)

Integrate both sides:

\[ \frac{y}{(13+x)^8} = C. \]

Solve for \(y\):

\[ \boxed{y = C(13+x)^8} \]

Same answer as Method 1. ✓

Example 3 — Tank Mixing Problem

Problem A 120-gallon tank initially contains 90 lb of salt dissolved in 90 gal of water. Brine containing 2 lb/gal of salt flows into the tank at a rate of 4 gal/min, and the well-stirred mixture flows out at a rate of 3 gal/min. How much salt does the tank contain when it is full?

Setting Up the IVP

Let \(A(t)\) denote the amount (in pounds) of salt in the tank at time \(t\) (in minutes). The basic balance equation is

\[ \frac{dA}{dt} = \text{(rate in)} - \text{(rate out)}. \]
Units guide Each rate has units \(\dfrac{\text{lb}}{\text{min}} = \dfrac{\text{lb}}{\text{gal}} \times \dfrac{\text{gal}}{\text{min}}\), i.e.  concentration × flow rate.

Rate in: The incoming brine has concentration 2 lb/gal and flows at 4 gal/min:

\[ \text{rate in} = 2 \times 4 = 8 \;\frac{\text{lb}}{\text{min}}. \]

Rate out: The outgoing concentration equals the tank concentration \(\dfrac{A}{V}\), and the outflow is 3 gal/min. The volume of liquid in the tank is

\[ V(t) = 90 + (4 - 3)t = 90 + t \quad\text{gal} \]

(since we gain a net 1 gal/min). Therefore

\[ \text{rate out} = \frac{A}{90+t}\cdot 3 = \frac{3A}{90+t}\;\frac{\text{lb}}{\text{min}}. \]

The tank is full (120 gal) when \(90 + t = 120\), i.e. \(t = 30\) min. So the IVP is:

\[ \frac{dA}{dt} = 8 - \frac{3A}{90+t}, \qquad A(0) = 90. \] Find \(A(30)\).

Solving the Linear ODE

Put into standard form \(\dfrac{dA}{dt} + P(t)\,A = Q(t)\):

\[ \frac{dA}{dt} + \frac{3}{90+t}\,A = 8. \]

Here \(P(t) = \dfrac{3}{90+t}\) and \(Q(t) = 8\).

Integrating factor:

\[ \mu(t) = e^{\int \frac{3}{90+t}\,dt} = e^{3\ln|90+t|} = |90+t|^3. \]

Since we only care about \(0 \le t \le 30\), we have \(90+t > 0\), so \(\mu(t) = (90+t)^3\).

Multiply the ODE by \((90+t)^3\):

\[ (90+t)^3\,\frac{dA}{dt} + 3(90+t)^2\,A = 8(90+t)^3. \]

Recognize the product rule:

\[ \frac{d}{dt}\!\left[(90+t)^3\,A\right] = 8(90+t)^3. \]

(Check: \(\frac{d}{dt}\bigl((90+t)^3 A\bigr) = (90+t)^3\frac{dA}{dt} + 3(90+t)^2 A\). ✓)

Integrate both sides (using the substitution \(u = 90+t\)):

\[ (90+t)^3\,A = \int 8(90+t)^3\,dt = 2(90+t)^4 + C. \]

Solve for \(A\):

\[ A = 2(90+t) + \frac{C}{(90+t)^3}. \]

Apply the initial condition \(A(0) = 90\):

\[ 90 = 2(90) + \frac{C}{90^3} = 180 + \frac{C}{90^3}. \] \[ C = (90 - 180)\cdot 90^3 = -90^4. \]

So the particular solution is

\[ A(t) = 2(90+t) - \frac{90^4}{(90+t)^3}. \]

Find \(A(30)\):

\[ A(30) = 2(120) - \frac{90^4}{120^3} = 240 - \frac{90^4}{120^3}. \]

Computing: \(90^4 = 65{,}610{,}000\) and \(120^3 = 1{,}728{,}000\), so

\[ A(30) = 240 - \frac{65{,}610{,}000}{1{,}728{,}000} = 240 - \frac{6561}{172.8} \approx 240 - 37.97 \approx 202.03. \]
\[ \boxed{A(30) = 240 - \frac{90^4}{120^3} \approx 202 \text{ lb of salt.}} \]

Key Ideas for Tank Mixing Problems

Remember
  • These problems lead to first order linear ODEs.
  • The fundamental equation is \(\dfrac{dA}{dt} = \text{rate in} - \text{rate out}\).
  • Use units to guide the setup: each rate should have units of (amount)/(time).

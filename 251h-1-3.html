Slope Fields

We consider a first-order ODE

\[\frac{dy}{dx} = f(x, y).\]

For a solution passing through \((x, y)\) in the plane, the DE gives the slope of that solution at that point. This is the key interpretation: \(f(x,y)\) is the rule for the slope of any solution passing through \((x,y)\).

Therefore, a graphical way to visualize and approximate various solution curves is to indicate the DE-provided slope at various points in the plane.

Through a representative collection of points \((x, y)\) in the \(xy\)-plane, we draw a short segment with the appropriate slope through each. This provides a slope field (or direction field), and solution curves will be tangent to these segments.

How it works: At each sample point \((x,y)\), evaluate \(f(x,y)\) to get a slope, then draw a short line segment with that slope. For example, at \((1,1)\) draw a segment with slope \(f(1,1)\); at \((2,1)\) draw a segment with slope \(f(2,1)\); and so on across the plane.

Slope Field Examples

The textbook (Figures 1.3.2 a–d) shows slope fields and solution curves for several autonomous ODEs:

  • \(y' = 2y\) — exponential growth
  • \(y' = 0.5\,y\) — slower exponential growth
  • \(y' = -y\) — exponential decay
  • \(y' = -3y\) — faster exponential decay
Key observation: Solution curves are tangent to slope field segments. For autonomous ODEs (where \(f\) depends on \(y\) only), the slope stays constant along horizontal lines — the slope depends only on the dependent variable.

A table of slopes can help with hand-drawing: for \(y' = 2y\), at \((0,1)\) the slope is \(2(1) = 2\); at \((1,1)\) the slope is still \(2\); at \((2,1)\) the slope is still \(2\). The slope is the same at every point with the same \(y\)-value.

Slope Fields for Physical Models

Recall some of the models we introduced earlier. Try visualizing their slope fields using the tools above (you'll need to choose values for the constants).

Newton's Cooling (constant ambient temperature \(A\))

\[\frac{dT}{dt} = k(A - T), \quad k > 0\]
Example values: \(k = 0.1\), \(A = 72\), giving \(\dfrac{dy}{dx} = 0.1(72 - y)\).

Torricelli's Law (leaking barrel; \(y(t)\) = depth of water)

\[\frac{dy}{dt} = -h\sqrt{y}, \quad h > 0\]
Example values: \(h = 1\), giving \(\dfrac{dy}{dx} = -\sqrt{y}\).

Logistic Population Model (\(M\) = carrying capacity)

\[\frac{dP}{dt} = kP(M - P), \quad k > 0\]
Example values: \(k = 0.01\), \(M = 100\), giving \(\dfrac{dy}{dx} = 0.01\,y(100 - y)\).
Note: The solution curves drawn by the plotting tools are generated by numerical ODE schemes (e.g., Euler's method), which we will discuss in Sections 2.4–2.6.

Motivation: Why Existence and Uniqueness Matter

For modeling, we want our ODEs to reflect physical reality:

  • When we set up an experiment, something happens — so we want our ODEs to have solutions (existence).
  • Experiments are repeatable — nature "knows" what happens. In deterministic systems there is no randomness, so we want exactly one solution for each initial condition (uniqueness).

Existence and Uniqueness Theorem

Theorem (Existence and Uniqueness)

Suppose that both the function \(f(x, y)\) and its partial derivative \(\dfrac{\partial f}{\partial y}(x, y)\) are continuous on some rectangle in the \(xy\)-plane that contains the point \((a, b)\) in its interior.

Then, for some open interval \(I\) containing the point \(a\), the initial value problem

\[\frac{dy}{dx} = f(x, y), \qquad y(a) = b\]

has one and only one solution that is defined on the interval \(I\).

What the theorem tells us: When we're in this "good situation" (both \(f\) and \(\frac{\partial f}{\partial y}\) continuous near \((a,b)\)), there is only one solution through \((a,b)\). Different solution curves cannot cross at that point.
Computing \(\frac{\partial f}{\partial y}\): Treat \(x\) as a constant and use the usual derivative rules to differentiate with respect to \(y\).

The hypotheses require both conditions:

  1. \(f(x,y)\) is continuous on a rectangle containing \((a,b)\), and
  2. \(\dfrac{\partial f}{\partial y}(x,y)\) is continuous on that same rectangle.

If either condition fails at the initial point, the theorem makes no guarantee — existence or uniqueness (or both) may fail.

Remark: The Interval May Be Small

The interval \(I\) on which the solution is guaranteed to exist can be small — we don't ever expect globally defined solutions \(y(x)\) defined for all \(-\infty < x < \infty\).

There are cases of "finite time blowup" where \(\displaystyle\lim_{x \to c} |y(x)| \to \infty\) for finite \(c\). (See the blowup example at the end of these notes.)

Nonexistence Example

Example

Show that the IVP

\[x\,\frac{dy}{dx} = 4y, \qquad y(0) = 3\]

has no solutions. Why doesn't the existence and uniqueness theorem apply?

Why there is no solution

Rewrite the ODE in standard form by dividing both sides by \(x\):

\[\frac{dy}{dx} = \frac{4y}{x}\]

Ask: what would the slope of a solution through \((0, 3)\) be? Substituting gives

\[0 \cdot \frac{dy}{dx} = 4(3) = 12 \quad \Longrightarrow \quad 0 = 12,\]

which is a contradiction. We cannot even draw a slope field segment through \((0, 3)\) — there is no solution through this point.

Why the theorem doesn't apply

In standard form, \(f(x,y) = \dfrac{4y}{x}\) and \(\dfrac{\partial f}{\partial y} = \dfrac{4}{x}\). Neither is defined at \(x = 0\), so neither is continuous on any rectangle containing \((0, 3)\). The hypotheses of the existence and uniqueness theorem are not satisfied.

Nonuniqueness Example: \(xy' = 4y\), \(y(0) = 0\)

Example

Show that the IVP

\[x\,\frac{dy}{dx} = 4y, \qquad y(0) = 0\]

has a 2-parameter family of solutions:

\[y = \begin{cases} C\,x^4 & x < 0 \\ D\,x^4 & x \ge 0 \end{cases}\]

Why doesn't the existence and uniqueness theorem apply?

Verification

For \(x < 0\): If \(y = Cx^4\), then \(\dfrac{dy}{dx} = 4Cx^3\), and \(x \cdot 4Cx^3 = 4Cx^4 = 4y\). ✓

For \(x \ge 0\): If \(y = Dx^4\), then \(\dfrac{dy}{dx} = 4Dx^3\), and \(x \cdot 4Dx^3 = 4Dx^4 = 4y\). ✓

The piecewise function is differentiable at \(x = 0\) because \(\dfrac{dy}{dx}\bigg|_{x=0} = 0\) from both sides (the slopes match), and \(y(0) = 0\). So this gives a valid solution for any choice of \(C\) and \(D\) — a 2-parameter family of solutions all passing through \((0, 0)\).

Why the theorem doesn't apply

Again, \(f(x,y) = \dfrac{4y}{x}\) and \(\dfrac{\partial f}{\partial y} = \dfrac{4}{x}\), and neither is continuous on any rectangle containing \((0, 0)\). The theorem's hypotheses fail.

Local vs. global uniqueness: Consider the related IVP with \(y(1) = 1\). Here the existence and uniqueness theorem does apply near \((1,1)\), since both \(f\) and \(\frac{\partial f}{\partial y}\) are continuous on any rectangle containing \((1,1)\) that avoids \(x = 0\). The initial condition \(y(1) = 1\) forces \(D = 1\), so locally near \(x = 1\) there is a unique solution \(y = x^4\). However, \(C\) is still free, so there are infinitely many global solutions. This is not a contradiction — the theorem only promises uniqueness on some small interval containing \(x = 1\).

Nonuniqueness Example: \(y' = y^{1/3}\), \(y(0) = 0\)

Example

Consider the IVP

\[\frac{dy}{dx} = y^{1/3}, \qquad y(0) = 0.\]

The trivial solution

Notice that \(y(x) = 0\) for all \(x\) is a solution, since \(\dfrac{dy}{dx} = 0\) and \(0^{1/3} = 0\). ✓

A nontrivial solution

Using separable-variable techniques (Section 1.4), one can find that

\[y(x) = \left(\frac{2}{3}\,x + C\right)^{3/2}\]

is also a family of solutions. Applying \(y(0) = 0\) gives \(C = 0\), so

\[y(x) = \left(\frac{2}{3}\,x\right)^{3/2}\]

also solves the IVP (on the interval \(x \ge 0\)).

We can verify: \(\dfrac{dy}{dx} = \dfrac{3}{2}\left(\dfrac{2}{3}x\right)^{1/2} \cdot \dfrac{2}{3} = \left(\dfrac{2}{3}x\right)^{1/2}\), and \(y^{1/3} = \left[\left(\dfrac{2}{3}x\right)^{3/2}\right]^{1/3} = \left(\dfrac{2}{3}x\right)^{1/2}\). ✓

Infinitely many solutions

The situation is even worse. For any \(a \ge 0\),

\[y(x) = \begin{cases} 0 & x \le a \\ \pm\left(\dfrac{2}{3}(x - a)\right)^{3/2} & x > a \end{cases}\]

also solves the initial value problem. Uniqueness has failed catastrophically — we have infinitely many solutions satisfying the initial condition.

Why the theorem doesn't apply

Here \(f(x,y) = y^{1/3}\), which is continuous for all \(x\) and \(y\). However,

\[\frac{\partial f}{\partial y} = \frac{\partial}{\partial y}\!\left(y^{1/3}\right) = \frac{1}{3}\,y^{-2/3} = \frac{1}{3\,y^{2/3}},\]

which is not continuous at \(y = 0\) (it blows up). Since the initial condition has \(y(0) = 0\), there is no rectangle containing \((0, 0)\) on which \(\frac{\partial f}{\partial y}\) is continuous. The uniqueness part of the theorem does not apply.

Physical relevance: This sort of nonuniqueness also occurs in Torricelli's law for leaking tanks. On the homework, you'll think about how nonuniqueness actually makes sense in this modeling situation.

Finite Time Blowup Example

Example (Problem 1.1.47 on Written HW 1)

Consider the IVP

\[\frac{dy}{dx} = y^2, \qquad y(0) = y_0.\]

We'll learn that the solutions are:

  • If \(y_0 = 0\): \(y(x) = 0\) for all \(x\) (constant function).
  • If \(y_0 \ne 0\):
    \[y(x) = \frac{1}{\dfrac{1}{y_0} - x}\]

When \(y_0 > 0\), this solution has a vertical asymptote at the positive \(x\)-value \(x = \dfrac{1}{y_0}\). The solution blows up in finite time.

Concrete example: \(y' = y^2\), \(y(0) = 1\)

The solution is \(y = \dfrac{1}{1 - x}\), which is only defined (and differentiable) on \(-\infty < x < 1\). The solution to the IVP cannot be defined for all \(x\).

Concrete example: \(y' = y^2\), \(y(0) = 1000\)

The solution is \(y = \dfrac{1}{\frac{1}{1000} - x}\), which blows up at \(x = \dfrac{1}{1000} = 0.001\).

Takeaway: The interval of existence promised by the theorem may not extend very far at all. Even for "nice" ODEs satisfying the theorem's hypotheses, solutions can blow up at a finite value of \(x\).
